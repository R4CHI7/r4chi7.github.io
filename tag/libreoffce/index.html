GSoC - Improve Usability of Personas - Introduction

03-06-2014
By Rachit Gupta

In Technology.

tags: gsoclibreoffcefoss

I got selected for 2014's Google Summer of Code to work for LibreOffice. I will be working on the Improve Usability of Personas idea this summer. As my proposal indicates, the task will be to improve the UI of the Personalization dialog and the implementation behind it. The Personalization dialog allows users to select and apply Mozilla's Personas to their copy of Libreoffice. The dialog can be accessed by Tools -> Options -> Personalization -> Select Theme.

The current implementation requires the user to browse through the themes in their browser, select the theme of their liking, copy and paste the URL in the dialog, and wait till the theme is downloaded and applied. Too much work IMO. My work during the summer will be to make this process easy and intuitive for the users. Also, the current implementation parses the HTML from the personas page to get the details. Following are the main sub tasks for this summer:

  • Allow users to search for themes in the dialog with the help of Mozilla's AMO API.
  • Allow users to select their favorite theme from the dialog itself.
  • Make the searching and downloading of themes run in a separate thread so the UI does not hang awkwardly.
  • Allow users(read designers) to apply their self made themes.
  • Solve some Personas related bugs/enhancement requests.
  • Possibly, do some pagination so the users can browse more themes in the dialog.

With some research, and a lot of help from my mentor, Jan Holesovsky, the work is under progress. Keep watching this space for more about the progress!

Cheers to the summer of 2014!
