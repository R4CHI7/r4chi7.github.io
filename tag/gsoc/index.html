GSoC - Improve Usability of Personas - Introduction

03-06-2014
By Rachit Gupta

In Technology.

tags: gsoclibreofficefoss

Libreoffice 4.0 introduced the concept of using Personas. The users can apply themes to their office suite! Since then, there has been an idea on the GSoC Page wiki to improve the usability of the selection of the themes process.

But, what improvements? The current implementation requires the user to browse through the themes in their browser, select the theme of their liking, copy and paste the URL in the dialog, and wait till the theme is downloaded and applied. Too much work IMO. My work during the summer will be to make this process easy and intuitive for the users. Also, the current implementation parses the HTML from the personas page to get the details.

My proposal for the idea got selected and I'll be working on this idea this summer. My aim is to make the process of searching for themes and applying them, easy and intuitive for the users, alongwith making the process work better and faster.

Following are the main tasks for this summer:

  • Allow users to search for themes in the dialog with the help of Mozilla's AMO API.
  • Allow users to select their favorite theme from the dialog itself.
  • Make the searching and downloading of themes run in a separate thread so the UI does not hang awkwardly.
  • Allow users(read designers) to apply their self made themes.
  • Solve some Personas related bugs/enhancement requests.
  • Possibly, do some pagination so the users can browse more themes in the dialog.

With some research, and a lot of help from my mentor, Jan Holesovsky, the work is under progress. Keep watching this space for more about the progress!

Cheers to the summer of 2014!
